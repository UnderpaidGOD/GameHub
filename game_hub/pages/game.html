Engineering the Logic Gate

The arcade modules presented here are built entirely using modern Vanilla JavaScript, DOM manipulation, and logic-based algorithms. Unlike the narrative-driven RPG or Open World pages, this arcade focuses on the core principles of game theory and event-driven programming. Each module acts as a "logic gate" where user clicks trigger state changes within the application, updating the interface in real-time without requiring a page refresh.

The Checkers AI utilizes basic minimax principles to anticipate player moves, while the UNO Match module handles complex deck-shuffling arrays and card-matching logic. Building these systems required a deep understanding of data structures—specifically arrays and objects—to track game states across different turns. This page demonstrates the "Functional" side of web development, where the code doesn't just look good, it acts and reacts to the player's every decision.
